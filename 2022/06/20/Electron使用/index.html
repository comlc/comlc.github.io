Electron使用
2022-06-20|前端Electron|
字数总计: 545|阅读时长: 2 分钟

简介

Electron是使用传统前端技术（Nodejs, Javascript, HTML, CSS）开发跨平台桌面应用（Windows、OSX及Linux）的框架。

Node.js与nvm、npm

nvm，全称是Node Version Manager，是用来管理node.js的版本的。

  1. 下载地址nvm-windows(nvm-setup.zip)

  2. 设置国内镜像：

    nvm node_mirror https://npm.taobao.org/mirrors/node/

    nvm npm_mirror https://npm.taobao.org/mirrors/npm/

  3. 由于不同的node项目所依赖的node版本可能不相同，通过nvm安装多个node版本，所以可以通过nvm来管理node的版本。

  4. 常用命令：

    安装最新的node.js版本：nvm install latest

    卸载某个node.js版本：nvm uninstall <version>

    查看当前nvm版本：nvm v

    安装的node.js版本列表：nvm list

    使用固定node.js版本：nvm use <version>

npm，全称是Node Package Manager，是以JavaScript编写的软件包管理工具，用来安装node项目所需要的一堆依赖包。安装完Node即顺带已安装好npm

  1. 设置国内镜像：npm config set registry https://registry.npm.taobao.org

  2. 修改本地仓库路径：

    npm config set prefix "D:\workspace\Repositories\Node\Global"

    npm config set cache "D:\workspace\Repositories\Node\Cache"

  3. 常用命令：

    查看配置信息：npm config ls

    查看版本信息：npm -v

    安装模块：npm install

    执行脚本：npm run

环境准备

Node.js推荐版本：^12.22.0 || ^14.17.0 || >=16.0.0，这里选择14.18.0安装

  1. 修改npm淘宝镜像：npm config set registry http://registry.npm.taobao.org/

  2. 修改electron淘宝镜像：npm config set electron_mirror https://npm.taobao.org/mirrors/electron/

  3. 全局安装yarn：npm install -g yarn

调用动态库ffi-napi支持

  1. 下载Python2.7.x版本，这里选择Python2.7.15安装
  2. 配置Python到npm：npm config set python "安装目录\python.exe"
  3. 安装node-gyp：npm install -g node-gyp
  4. 若本地已安装Visual Studio，则配置到npm：npm config set msvs_version 自己VS版本
  5. 若没有安装Visual Studio，则全局安装：npm install --global windows-build-tools

项目编译

  1. 执行yarn install安装依赖包
  2. 执行yarn run dev开发模式运行项目
  3. 其他常用命令
    • yarn build打包
    • yarn build:clean清理构建文件
    • yarn pack:resources打增量包，用于增量更新，需在yarn build后执行
